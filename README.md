# jQuery Pins

## Description

A jQuery plugin to display posts in a pin board style.

## Examples

Live examples coming soon...

## Useage

### HTML
```html
<div id="posts">
    <div class="pin">
        <p>
            Lorem ipsum
        </p>
    </div>
    <div class="pin">
        <p>
            Dolor sit
        </p>
    </div>
    ...
</div>
```

### jQuery
```js
$('#posts').pins();
```
